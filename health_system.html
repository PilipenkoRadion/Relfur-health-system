Systemarchitektur

Input Layer:
📷 Webcam / Kamera (640x480 @ 30 FPS)
↓
Processing Layer (Python Backend):
├─ 🎥 OpenCV: Videoaufnahme
├─ 🧠 MediaPipe: Skelett + Hand-Tracking
├─ 📊 NumPy FFT: BPM-Analyse (RPPG)
├─ 🔥 Kalorie-Engine: MET-Berechnung
├─ ⚡ FastAPI: REST API + WebSocket
└─ 📁 PyQt5: Desktop Monitoring
↓
Communication Layer:
🔌 WebSocket (Echtzeit-Daten)
📡 REST API (Status & Konfiguration)
↓
Output Layer (Frontend):
🌐 HTML5/CSS/JavaScript Web-Interface
📺 Canvas: Video & Skelett-Rendering
📊 Live-Grafiken & Leaderboard
↓
Storage:
💾 JSON Records Database

Datenfluss: Kamera → OpenCV/MediaPipe → NumPy Analyse → FastAPI WebSocket → Frontend Canvas Rendering
